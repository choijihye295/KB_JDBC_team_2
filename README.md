# KB_JDBC_team_2 : 마이데이터 통합 자산관리 시스템 

## 🎯 프로젝트 개요
본 프로젝트는 여러 금융 도메인(계좌, 거래내역, 대출, 보험)의 데이터를 연동하는 시스템입니다. 

---

## 🛠️ Tech Stack
- **Language**: Java 17
- **Database**: Local MySQL
- **Library / Tool**: JDBC, Lombok, Gradle

---

## 🚦 핵심 기능 명세 (Core Features)

본 시스템은 로그인한 사용자의 고유 세션(`MemberVO`)을 기반으로 금융 도메인 전반을 유기적으로 연결합니다.

|  | 기능명 | 상세 역할 및 데이터 연동 범위                                                             |
| :---: | :--- |:------------------------------------------------------------------------------|
| **1** | **계좌 조회** | - 사용자가 보유한 금융권 전체 계좌 목록 및 잔액 출력<br>- 개별 계좌 선택 시 계좌 종류 및 개설일 등 상세 정보 제공        |
| **2** | **거래내역 조회** | - 입출금 거래 내역 전체 로드<br>- 계좌번호, 기간(시작/종료일), 유형(입/출금), 키워드 조건별 동적 검색 지원           |
| **3** | **소비 분석** | - 카테고리별(식비, 쇼핑, 교통비 등) 지출 통계<br>                                              |
| **4** | **자산 관리** | - 유저의 예적금 잔액, 대출 잔액 등을 종합한 총자산 현황 파악<br>                                      |
| **5** | **대출 및 보험 조회** | - 대출: 총 대출 금액 합계 및 기관별 대출 상품 계약 정보 조회<br>- 보험: 가입 보험 목록, 월 납입금 현황 및 만기일 통합 조회 |
| **6** | **로그아웃** | - 사용자 고유 번호 및 로그인 세션 객체 초기화 후 로그인 전 메인 화면 복귀                                  |

---

- **팀장** : 안효선
- **팀원** : 최지혜, 허현정, 조윤상, 이성우
